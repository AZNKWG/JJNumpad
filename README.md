## Bill of Materials:
- 1 Seeed Studio XIAO RP2040
- 17 Gateron KS-8 Switches
- 17 1N4148 Diodes
- 4x M3x16 Bolt
- 4x M3 Heatset
- 1 17-Key Key Cap Set
